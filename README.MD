# Minecraft Emoji Plugin
This plugin allows you to use emojis in Minecraft chat.

## How it works
A custom font is used inside a texture pack to get the emojis. Each emoji is a custom unicode character with a custom texture. When you type the name of an emoji in the chat, for example `:flushed:` the plugin will replace it with the custom unicode character so it will become `ðŸ˜³`

## Commands
| Command Name  | Description                        |
| ------------- | ---------------------------------- |
| `/emoji`      | Plugin information and help        |
| `/emoji list` | List of all emojis and their names |
